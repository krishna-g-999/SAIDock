About SAIDock

Overview

SAIDock is an automated multi-stage pipeline for structure-based drug-target interaction assessment. It integrates pocket detection, molecular docking, ADMET profiling, ML scoring, and literature validation into a single reproducible workflow. The DTSS ranks compound-target pairs beyond raw docking affinity alone.

Key Capabilities

  • Automated multi-pocket detection and druggability classification
  • AutoDock Vina docking across all detected pockets
  • 19-parameter ADMET profiling via RDKit descriptors
  • Random Forest ML scoring with applicability domain
  • Real-time ChEMBL literature prior integration
  • DTSS composite scoring for lead prioritisation
